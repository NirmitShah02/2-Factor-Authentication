

## Setup

1- Clone the project:
```
https://github.com/NirmitShah02/2-Factor-Authentication.git
```
2- Open the folder:

```
cd FaceRecognitionAuth
```
3- Install dependencies:

```
flutter pub get
```
Run in iOS directory
```
pod install
```
4- Run on device (Check device connected or any virtual device running):

```
flutter run
```
